# People Directory

Un fichier markdown par personne. Format simple :

```markdown
# Nom Complet

- **Relation:** comment tu connais cette personne
- **Contexte:** où vous interagissez
- **Contact:** email, téléphone, etc.

## Notes

Points clés à retenir.

## Historique

- YYYY-MM-DD : événement notable
```

Pas de JSON, pas de schema compliqué. Juste du markdown lisible.

## Qui devrait avoir un fichier ?

- Contacts professionnels récurrents
- Collaborateurs sur des projets
- Prospects/clients importants
- Pas les one-shot interactions
