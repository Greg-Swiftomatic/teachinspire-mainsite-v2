## 2. PROBLÈME

### Titre de section
```
Vos formateurs entendent parler d'IA partout.
Et après ?
```

### Corps
```
Certains bricolent avec ChatGPT — résultats aléatoires.
D'autres attendent, sceptiques ou débordés.
Pendant ce temps, d'autres instituts forment déjà leurs équipes.

Le vrai risque ? Ce n'est pas que l'IA remplace vos formateurs.
C'est qu'ils se retrouvent largués — pendant que d'autres prennent de l'avance.

Trois scénarios à éviter :

• Dépendants d'outils qu'ils ne maîtrisent pas
• Incapables de s'adapter quand l'outil change
• Dépassés par ceux qui ont compris la méthode

L'IA évolue vite. Les outils d'aujourd'hui seront obsolètes demain.
Ce qui reste ? La capacité à penser l'IA.
```

## 3. SOLUTION / APPROCHE

### Titre de section
```
On n'enseigne pas un outil.
On enseigne à penser l'IA.
```

### Corps
```
La plupart des formations IA vous apprennent à utiliser ChatGPT.
Problème : quand ChatGPT change (ou qu'un meilleur outil arrive), 
vous repartez de zéro.

TeachInspire enseigne une méthode transférable :

✓ Évaluer n'importe quel outil IA
✓ Comprendre ses forces et limites
✓ L'adapter à son contexte pédagogique
✓ Garder le contrôle sur sa pratique

Vos formateurs deviennent autonomes — pas utilisateurs captifs.
```

### Titre de section
```
Un parcours structuré pour vos équipes
```

### Format (4 étapes)

**Étape 1 : Appel découverte**
```
On échange sur vos besoins, votre contexte, vos objectifs.
Pas de pitch commercial — juste une conversation.
```

**Étape 2 : Formation hybride**
```
15 modules vidéo (20 min chacun) + 6-8 sessions live sur 12 semaines.
Vos formateurs avancent à leur rythme, avec un point toutes les 2 semaines.
```

**Étape 3 : Mise en pratique**
```
Pas de théorie abstraite. Chaque module se termine par 
une application concrète avec leurs vrais apprenants.
```

**Étape 4 : Communauté**
```
Accès à la communauté TeachInspire pendant 1 an.
Questions, mises à jour sur les outils, entraide entre instituts.
```

### Note sur le format
```
Format groupe (10-25 formateurs) pour que toute l'équipe avance ensemble.
Espace dédié pour votre institut dans la communauté.
```

## 9. FAQ / OBJECTIONS

### Titre de section
```
Questions fréquentes
```

### Q1 : Mes formateurs ne sont pas à l'aise avec la tech
```
R : La formation est progressive et part de zéro. 
Pas besoin de compétences techniques préalables. 
On utilise des outils conçus pour être accessibles.
```

### Q2 : Ça va coûter cher en abonnements ?
```
R : Non. On utilise principalement Google AI Studio — 
gratuit et quasi-illimité, même pour la synthèse vocale pro. 
Pas d'abonnements qui s'empilent.
```

### Q3 : J'ai testé ChatGPT, les résultats étaient moyens
```
R : Normal. Sans méthode structurée, les résultats sont aléatoires.
C'est précisément ce qu'on enseigne : comment obtenir 
des résultats consistants, pas des coups de chance.
```

### Q4 : On n'a pas le temps
```
R : Les modules font 20 minutes et s'intègrent dans le planning.
Format pensé pour des formateurs en activité, pas des étudiants à temps plein.
```

### Q5 : Et si les outils changent ?
```
R : Ils vont changer, c'est certain. C'est pour ça qu'on enseigne 
une méthode transférable, pas un mode d'emploi spécifique.
Vos formateurs sauront s'adapter.
```

### Q6 : C'est finançable par OPCO ?
```
R : Oui, la formation est éligible au financement OPCO.
On vous accompagne dans les démarches si besoin.
```

### Q7 : Combien ça coûte ?
```
R : Le tarif dépend de la taille de votre équipe.
Standard (≤10 formateurs) : à partir de 4 200€ HT.
On en parle lors de l'appel découverte — sans engagement.
```

### Q8 : Et si ça ne correspond pas à nos besoins ?
```
R : On le saura dès l'appel découverte. 
Pas de pression, pas de relance. Si ce n'est pas un fit, on vous le dira.
```
